# Networking Module

```text
Type: Module
Difficulty: Easy
Description:
Learn about the OSI model and TCP/IP networking layers. Explore the different plaintext and secure 
networking protocols that we use every day.

This module aims to introduce you to the ISO OSI and TCP/IP models and present all the main protocols 
necessary for networks to function. It also introduces all the essential protocols for web browsing, 
file transfer, email, and remote administration.
```
