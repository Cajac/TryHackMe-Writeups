# Web Hacking Module

```text
Type: Module
Difficulty: Easy
Description:
Learn about web applications, JavaScript, and SQL. Explore BurpSuite, a web 
application security testing platform, and the OWASP Top Ten.

This module helps the user become familiar with web applications, JavaScript, 
database systems, and SQL. Furthermore, it teaches the basics of BurpSuite 
so the user gains the necessary knowledge to conduct various web application 
security tests. Finally, it covers the OWASP Top 10 web application security 
risks.
```
