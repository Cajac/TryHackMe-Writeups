# Wireshark Module

```text
Type: Module
Difficulty: Easy
Description:
Wireshark is the industry-standard tool for network protocol analysis and is essential in 
any kind of traffic investigation.

This module will cover need-to-know functionalities of Wireshark for any security analyst: 
Traffic Sniffing, Packet Dissection & Navigation, Packet Filtering, and Traffic Analysis. 
You will learn how to use Wireshark when investigating live and captured traffic and 
identify anomalies, threats and attacks. By the end of the module, you will master your 
Wireshark skills and be able to investigate complex network cases and create IOC lists 
and actionable results with Wireshark.
```
