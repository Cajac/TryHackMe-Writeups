# Defensive Security Module

```text
Type: Module
Difficulty: Easy
Description:
This module will help you build your defensive security knowledge. It covers key 
topics such as SOC, digital forensics, and incident response.

This module will teach you various critical cyber security skills for effectively 
monitoring, detecting, and responding to security incidents. It will also explain 
the role of the Security Operations Center (SOC) team and the tools they use. 
Furthermore, it will help you acquire essential digital forensics and incident 
response techniques. This knowledge will help you build a solid foundation in 
defensive security.
```
